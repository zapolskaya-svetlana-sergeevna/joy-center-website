Живая ссылка на работающий проект: [joy-center.infinityfree.me](http://joy-center.infinityfree.me)

Данный проект представляет собой многостраничный веб-сайт для психологического центра "J.O.Y.". Сайт полностью работоспособен, интегрирован с базой данных для хранения пользовательских данных (записи на прием, формы обратной связи) и развернут на хостинге InfinityFree.

* **Frontend:** HTML5, CSS3, JavaScript
* **Backend:** PHP
* **База данных:** MySQL (дамп базы данных прикреплен в корне репозитория файлом `.sql`)

1. **Разработка:** Написан адаптивный интерфейс сайта, настроена отправка форм и логика взаимодействия с сервером на стороне PHP.
2. **Работа с БД:** Спроектирована реляционная база данных для хранения заявок клиентов и тд, настроено безопасное подключение.
3. **Развертывание и внедрение:**
   * Выбран и настроен хостинг InfinityFree.
   * Настроены DNS-записи для домена.
   * База данных импортирована на удаленный сервер.
   * Проведена диагностика работы скриптов и отладка сетевых запросов.
